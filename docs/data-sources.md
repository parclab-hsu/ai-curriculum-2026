> 본 사이트 여러 트랙 페이지의 「채용 동향 — 거시 지표」에서 반복 인용되는 **국내 AI 채용 지표**의 1차 출처·조사 시점·표본·정의를 한 곳에 정리한다. 각 트랙의 수치는 이 표를 근거로 한다.
> 인용 형식: **기관 — 「조사·보고서명」 (발표일) · URL** / 확인일 2026-06-27

!!! warning "해석 주의"
    각 조사는 기관·시점·표본·직무 정의가 달라 **직접 비교에 한계**가 있다. 아래 수치는 발표·언론 보도 기준이며, 인용 시 1차 출처 원문을 함께 확인할 것. **잡코리아·대한상의의 보도자료 원문(1차) URL을 확보하지 못한 항목은 아래 '재인용(언론)' 링크가 1차가 아닌 언론 재인용 기준임에 유의한다.**

## 5. 채용공고 스냅샷 재현성 기준 (Fact Check 2026-07 반영)

!!! warning "실시간 채용 플랫폼 수치의 한계"
    사람인·잡코리아·원티드·LinkedIn 등 **실시간 채용 플랫폼의 개별 공고 수**는 로그인·WAF·동적 렌더링·조회 시점에 따라 달라진다. 따라서 각 트랙 페이지의 '채용공고 건수'는 **독립 재집계가 아닌 출처 재현성 기준**으로 다루며, 아래 항목을 함께 기록해야 재현·검증이 가능하다.

각 트랙 페이지에서 채용공고 건수를 인용할 때는 **다음 6개 항목을 출처 하단에 명시**한다(값이 없으면 '방향성 참고'로만 인정).

| 기록 항목 | 설명 | 예시 |
|---|---|---|
| **검색어(쿼리)** | 실제 입력한 키워드·불리언 식 | `"생성형 AI" AND (디자인 OR 콘텐츠)` |
| **플랫폼** | 조회한 채용 플랫폼 | 사람인 / 잡코리아 / 원티드 / LinkedIn |
| **필터** | 지역·경력·고용형태·직무분류 등 | 신입 · 전국 · 정규직 |
| **조회일(스냅샷 일시)** | 조회한 날짜·시각(KST) | 2026-07-01 10:00 KST |
| **결과 건수** | 조회 시점의 공고 수 | 1,240건 |
| **캡처/CSV 위치** | 화면 캡처 또는 CSV 보존 경로 | `data/snapshots/2026Q3/design_saramin.csv` |

> **권고 운영:** 위 스냅샷을 **매 학기 CSV/캡처로 보존**해 연도·학기 비교가 가능한 데이터셋으로 축적한다. 보존 저장소 예: GitHub `parclab-hsu/tech-trends-2024-2026` 의 `data/snapshots/`.

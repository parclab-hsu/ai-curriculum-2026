# 한성대학교 교육혁신센터 · AI융합교육과정 분석

이 사이트는 **2026학년도 AI융합교육과정 개편** 대상 트랙·학과를 대상으로, **기술 트렌드**와 **직무 역량 수요**를 데이터로 분석하고 트랙별 교육 목표·모듈형 전공교육과정까지 제안한 리포트입니다. 채용·산업 동향 데이터는 **LinkedIn·사람인·잡코리아·원티드** 등 실제 채용 플랫폼과 산업 보고서를 기반으로 합니다.

!!! note "한눈에 보기"
    기술은 `생성형 AI(2024) → 에이전트 AI(2025) → Physical AI(2026)`로 이동하고, 직무 시장은 `AI 활용 역량`을 가장 가파른 기울기(+11.9%p/년)로 요구합니다. 본 분석은 이 흐름을 각 트랙·학과의 **직무 역량 + AI 융합 + 교육과정 모듈**로 구체화합니다.

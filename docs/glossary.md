# 용어집 (Glossary)

> 본 사이트 트랙 분석·모듈형 교육과정에 등장하는 핵심 용어를 **설명 + 예제**와 함께 정리했습니다. 분야별로 묶었으며, 같은 용어가 여러 트랙에서 쓰일 수 있습니다. 확인일 2026-06-30.

## 1. AI·머신러닝 핵심

| 용어 (영문) | 설명 | 예제 |
| --- | --- | --- |
| 인공지능 (AI, Artificial Intelligence) | 학습·추론·인식 등 인간 지능을 컴퓨터로 구현하는 기술의 총칭 | 이미지 분류, 음성 비서, 추천 시스템 |
| 머신러닝 (ML, Machine Learning) | 데이터로부터 규칙을 스스로 학습하는 AI 방법론 | 스팸 메일 자동 분류 모델 학습 |
| 딥러닝 (Deep Learning) | 다층 인공신경망 기반 머신러닝 | CNN으로 사진 속 객체 검출 |
| 지도/비지도 학습 (Supervised/Unsupervised) | 정답 라벨 유무에 따른 학습 방식 | (지도) 개·고양이 분류 / (비지도) 고객 군집화 |
| 강화학습 (RL, Reinforcement Learning) | 보상을 최대화하도록 시행착오로 행동을 학습 | 로봇 보행·집기 정책 학습 |
| 컴퓨터비전 (CV, Computer Vision) | 이미지·영상을 인식·이해하는 AI | 차선 인식, 불량 검사 |
| MLOps | ML 모델의 학습·배포·모니터링·재학습을 자동화하는 운영 체계 | 모델 버전관리·서빙·성능 모니터링 파이프라인 |
| 파인튜닝 · LoRA (Fine-tuning) | 사전학습 모델을 특정 과제에 맞게 추가 학습(LoRA=경량 기법) | 사내 문서로 LLM 미세조정 |

## 2. 생성형 AI·LLM

| 용어 (영문) | 설명 | 예제 |
| --- | --- | --- |
| 생성형 AI (Generative AI) | 텍스트·이미지·영상 등 새 콘텐츠를 만들어내는 AI | 이미지·카피 시안 자동 생성 |
| 거대언어모델 (LLM, Large Language Model) | 방대한 텍스트로 학습한 언어 생성 모델 | GPT, HyperCLOVA X, EXAONE |
| 에이전트 AI (Agentic AI) | 스스로 계획·도구 사용·실행하는 자율 AI | "항공권 찾아 예약"까지 수행하는 AI 비서 |
| RAG (Retrieval-Augmented Generation) | 외부 지식을 검색해 LLM 답변에 결합하는 기법 | 사내 규정을 검색 후 답변하는 챗봇 |
| 벡터DB (Vector Database) | 의미 임베딩을 저장하고 유사도로 검색하는 DB | Pinecone·FAISS로 문서 의미 검색 |
| 프롬프트 엔지니어링 (Prompt Engineering) | 원하는 출력을 끌어내도록 입력을 설계하는 기술 | 역할·예시·제약을 담은 프롬프트 작성 |
| 멀티모달 · 옴니모달 (Multimodal/Omnimodal) | 텍스트·이미지·음성 등 복수 양식을 함께 처리 | 사진과 질문을 같이 입력해 답변 |
| VLA (Vision-Language-Action) | 시각·언어·행동을 잇는 로봇·모빌리티용 모델 | "빨간 컵 집어줘" 명령을 수행 |

## 3. Physical AI·로봇·모빌리티

| 용어 (영문) | 설명 | 예제 |
| --- | --- | --- |
| Physical AI | 물리 세계에서 인지·추론·계획·행동을 수행하는 체화형 AI | 자율주행차, 휴머노이드 |
| Embodied AI (체화 AI) | 로봇 등 '몸'을 통해 환경과 상호작용하는 AI | 물류 창고 피킹 로봇 |
| 휴머노이드 (Humanoid) | 사람 형태의 범용 로봇 | Tesla Optimus, Boston Dynamics Atlas |
| ROS · ROS2 (Robot Operating System) | 로봇 소프트웨어 표준 미들웨어 | 센서·제어 노드 간 통신 |
| SLAM | 동시적 위치추정·지도작성 | 로봇청소기의 실내 지도 생성 |
| Sim-to-Real | 시뮬레이션에서 학습한 정책을 실제 로봇으로 이전 | Isaac Sim 학습 → 실로봇 적용 |
| Edge AI · 온디바이스 AI | 클라우드 없이 기기 자체에서 추론 | 스마트폰 내 실시간 음성 인식 |
| SDV (Software Defined Vehicle) | 소프트웨어로 기능이 정의·업데이트되는 차량 | OTA로 기능이 추가되는 전기차 |
| AUTOSAR | 차량 전장 SW 표준 아키텍처 | ECU 소프트웨어 표준화 |
| UAM · eVTOL | 도심항공모빌리티 · 전기수직이착륙기 | 도심 에어택시 |

## 4. 반도체·임베디드·기계공학

| 용어 (영문) | 설명 | 예제 |
| --- | --- | --- |
| NPU (Neural Processing Unit) | AI 연산 전용 가속기 | 온디바이스 AI 추론 칩 |
| TinyML | 초저전력 기기에서 동작하는 경량 ML | MCU에서 키워드 인식 |
| HBM (고대역폭 메모리) | AI 가속기용 고속 적층 메모리 | GPU 옆 스택 메모리 |
| EDA (전자설계자동화) | 반도체 설계를 자동화하는 도구 | RTL→레이아웃 설계 흐름 |
| RTOS (실시간 운영체제) | 정해진 시간 내 응답을 보장하는 OS | 차량·로봇 실시간 제어 |
| 4대 역학 | 정역학·동역학·열역학·유체역학(기계공학 기초) | 구조·운동·열·유체 해석 |
| CAE · FEA · CFD | 전산해석 · 유한요소해석 · 전산유체역학 | 부품 강도·열·유동 시뮬레이션 |
| 디지털트윈 (Digital Twin) | 실물을 가상으로 복제해 시뮬레이션 | 가상 창고로 물류 동선 최적화 |
| 센서퓨전 (Sensor Fusion) | 여러 센서 데이터를 융합해 인식 정확도를 높임 | 카메라+라이다 융합 인지 |

## 5. 디자인·콘텐츠

| 용어 (영문) | 설명 | 예제 |
| --- | --- | --- |
| UX · UI | 사용자 경험 · 사용자 인터페이스 | 앱 화면·플로우 설계 |
| BX (Brand Experience) | 브랜드 경험 디자인 | 캠페인·브랜드 세계관 |
| VMD (Visual Merchandising) | 시각적 상품 연출 디자인 | 매장 디스플레이·전시 연출 |
| 버추얼 프로덕션 (Virtual Production) | 실시간 엔진 기반 영상 제작 | LED월+언리얼엔진 촬영 |
| Gaussian Splatting | 3D 장면을 점군으로 실시간 렌더링하는 신기술 | 실사 공간의 3D 복원 |
| 디자인 토큰 (Design Tokens, DTCG) | 색·간격 등 디자인 값을 표준화한 변수 — **W3C Community Group(DTCG) 사양으로, W3C 공식 표준은 아님** | 디자인-코드 일관성 유지 |
| 프로비넌스 (Provenance) | 콘텐츠의 출처·생성 이력 증명 | AI 생성물의 출처 표기 |
| 테크팩 (Tech Pack) | 의류 생산 사양서 | 패턴·소재·치수 명세서 |

## 7. 교육과정·문서 표기 용어

| 용어 | 설명 | 예제 |
| --- | --- | --- |
| 모듈형 교육과정 | 3~4개 교과목을 역량 단위로 묶은 구성 | 'AI 핵심·머신러닝' 모듈(인공지능 개론 등 3~4과목) |
| 캡스톤디자인 (Capstone) | 전공을 종합하는 졸업 프로젝트 | 자율주행 인지→제어 통합 프로젝트 |
| 마이크로디그리 (Micro-degree) | 특정 모듈 이수 인증 | 'Sim-to-Real' 마이크로디그리 |
| (공식) 표기 | 한성대 **공식 교과과정**에 실재하는 과목 | 자율주행시스템 응용(공식) |
| (제안) · (미정) 표기 | (제안)=공식 목록에 없는 **신설 제안 교과**(새로 만들지 않고 표시), (미정)=개설 학년·학기 미정 | RAG시스템설계(제안) |
| reported · estimated | 출처 보고값 · 인접연도·성장률로 역산한 추정값 | 차트의 진한/흐린 마커 구분 |
